## About

**Firefly AI Toolkit** gives you instant access to generative assets. Activate **Image Model 5**, **Video Generation**, **Audio Tools**, load **prompts and presets**, and use **app configs** — all in one click for seamless workflows on Apple Silicon.

---

## What's included

- `presets/` — Image/video/audio prompts, styles
- `models/` — Firefly Image 5, partner integrations
- `configs/` — Photoshop, Illustrator profiles
- `guides/` — 30-second setup, Apple Silicon tips
- `assets/` — UI themes, generation templates

## Download & install (macOS)

1. Click **Launch Firefly AI Now**
2. Extract to `~/Applications/Firefly-Toolkit`.
3. Import into Firefly/Photoshop → assets auto-load.
> Pro tip: Use GPU acceleration for M1/M4 performance.
